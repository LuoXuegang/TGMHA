水质预测系统
项目概述
本项目是一个综合性的水质监测与预测系统，利用多种时间序列预测模型对水质参数进行分析和预测。系统集成了数据处理、探索性分析、模型训练、评估和可视化等功能，旨在提供准确的水质趋势预测，辅助水质监测和管理决策。
数据集介绍
数据来源
本系统使用的数据集来源于四川省的水质监测站点，包含了多个监测点在不同时间的水质参数记录。原始数据存储在data/raw/sichuan_data.csv文件中。
数据结构
数据集主要包含以下字段：

断面名称：监测站点的名称，如"201医院"、"三块石"、"岷江大桥"等
监测时间：数据记录的时间点
水质参数：包括多种水质指标，主要有：

水温(℃)：水体温度
pH(无量纲)：水体酸碱度
溶解氧(mg/L)：水中溶解的氧气含量
电导率(μS/cm)：反映水中离子总含量
浊度(NTU)：水的浑浊程度
高锰酸盐指数(mg/L)：反映水中有机物含量
氨氮(mg/L)：水中氨和铵离子含量
总磷(mg/L)：水中总磷含量
总氮(mg/L)：水中总氮含量
叶绿素α(mg/L)：反映水中藻类含量
藻密度(cells/L)：水中藻类细胞密度



数据特点

时间序列特性：数据按时间顺序记录，适合进行时间序列分析
多站点覆盖：包含103个不同监测站点的数据
多参数监测：同时监测多种水质参数，便于综合分析
数据不均衡：不同站点的数据量存在差异，部分站点数据较少
季节性变化：数据表现出明显的季节性和周期性特征
异常值和缺失值：存在一定比例的异常值和缺失值，需要进行数据清洗

数据预处理
系统对原始数据进行了以下预处理：

缺失值处理：使用前向填充、后向填充和中位数填充等方法处理缺失值
异常值处理：基于IQR（四分位距）方法识别和处理异常值
数据格式转换：将日期转换为标准的datetime格式，并设置为索引
时间序列重构：按站点和特征重构数据，便于时间序列分析
数据分割：将数据按8:1:1的比例分割为训练集、验证集和测试集

数据统计
数据集统计信息概览：

站点数量：约100个监测站点
时间跨度：根据站点不同，一般为近几年的监测记录
记录总数：约数万条记录
数据维度：每条记录包含11个主要水质参数指标
采样频率：不同站点采样频率不同，多为每月或每季度采样

模型与功能
系统实现了多种预测模型：

ARIMA/SARIMA：传统的统计学时间序列预测模型
ARIMA-ANN：结合ARIMA和神经网络的混合模型
CNN-LSTM：使用卷积和长短期记忆网络的深度学习模型
CNN-GRU-Attention：带有GRU和注意力机制的高级模型，支持多步预测
XGBoost：基于梯度提升的时间序列预测模型

使用说明

数据准备：将原始数据放入data/raw目录
运行系统：执行main.py文件启动系统
模型训练：使用model_training.py训练特定模型
查看结果：通过UI界面或查看results目录中的可视化结果

示例命令：
python main.py --mode all --models cnn_gru_attention xgboost --top_n_stations 5 --forecast_horizon 7
系统目录结构

data/：数据存储目录

raw/：原始数据
processed/：处理后的数据


models/：模型实现
results/：结果输出

figures/：图表和可视化
metrics/：评估指标


log/：日志文件
